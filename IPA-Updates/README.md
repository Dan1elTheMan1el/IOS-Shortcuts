Get notified on discord when an Altstore repo is updated!

# IPA Updates
## Features
- Select installed apps from any repo
- Automatically checks each repo for latest version of app
- Sends discord notification with download link and automatically tracks installed versions
- Ignore updates for individual apps
- Customize Webhook username/avatar

## Setup
- In your discord server / text channel, create a webhook and copy the URL
- Run **IPA Updates**, and enter the webhook URL in settings
- Setup automation as shown (text -> run shortcut)
- Copy a repo link and add apps to track

Enjoy :)
