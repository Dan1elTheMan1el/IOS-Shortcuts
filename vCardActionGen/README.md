Quickly copy/paste all vCard menu actions!

**Note: Requires [Actions](https://apps.apple.com/ca/app/actions/id1586435171) app**

# Features
- Clean UI to edit full menu
  - Add/remove items
  - Upload icons from photos/files
  - Create icons with SFSymbols
    - Customize background/foreground colors
  - Set title/subtitle
- Generates and copies all necessary actions
  - Base64 placed in separate dictionary
  - Adds text, set name, and choose from list
  - Adds If statements for each item included
  - Ready to paste in any shortcut instantly

Enjoy :)
